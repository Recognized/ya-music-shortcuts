### Глобальные шорткаты для Яндекс.Музыки™

Работают при свернутом окне Chrome.

- **Ctrl**+**Shift**+**1** Предыдущий трек 
- **Ctrl**+**Shift**+**2** Пауза/снять с паузы 
- **Ctrl**+**Shift**+**3** Следующий трек
